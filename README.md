# events-pipes

## Build it and run it

- Clone the repo
- lein cljsbuild once
- lein uberjar
- java -jar target/events-pipes-0.1.0-SNAPSHOT-standalone.jar


## Develope it

- lein figwheel (will give you a cljs repl and start an nrepl server)
- Connect to the nrepl server and run the main function
